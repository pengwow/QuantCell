# 新闻分析技能

## 用途

分析财经新闻对加密货币市场的潜在影响，帮助做出交易决策。

## 使用方法

1. 使用 `get_news` 工具获取最新新闻
2. 分析新闻内容，识别关键事件
3. 使用 `get_market_sentiment` 获取市场情绪
4. 综合判断市场走向

## 分析框架

### 利好信号
- 机构采用加密货币
- 监管政策明朗化
- 技术突破或升级
- 主流支付集成

### 利空信号
- 监管打击
- 安全漏洞
- 主要交易所问题
- 宏观经济恶化

## 示例

用户: "分析今天比特币相关新闻"

你应该:
1. 调用 get_news 查询 bitcoin
2. 分析每条新闻的影响
3. 结合市场情绪给出判断
