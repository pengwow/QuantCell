# 技术分析技能

## 用途

使用技术指标分析市场趋势，识别交易机会。

## 使用方法

1. 使用 `get_klines` 获取 K 线数据
2. 计算技术指标（MA, RSI, MACD 等）
3. 识别支撑阻力位
4. 给出交易建议

## 常用指标

### 移动平均线 (MA)
- MA5、MA10、MA20、MA60
- 金叉（短期上穿长期）：买入信号
- 死叉（短期下穿长期）：卖出信号

### 相对强弱指数 (RSI)
- RSI > 70：超买，可能回调
- RSI < 30：超卖，可能反弹

### MACD
- DIF 上穿 DEA：买入信号
- DIF 下穿 DEA：卖出信号

## 示例

用户: "分析 BTCUSDT 的技术面"

你应该:
1. 获取 1h 和 4h K 线数据
2. 计算关键指标
3. 分析趋势和关键价位
4. 给出操作建议
